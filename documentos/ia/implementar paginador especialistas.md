# Paginador de especialistas

## Descripción

Se requiere un paginador en el dateform que permita a los usuarios navegar entre los especialistas de manera cómoda y rápida.

## Funcionalidades

- Cargar los especialistas de manera paginada.
- Resaltar la página actual.
- Mostrar los botones de navegación.

## Diseño

Debe utilizar los colores y estilos de '/public/css/bootstrap-landing-theme.css'

## Otras consideraciones

- Debe de utilizar solo clases de bootstrap
- Debe usar latte para renderizar el html
- Debe de usar las directivas de latte y solo si es necesario php puro
- No debe de tener codigo muy complejo
- Debe de adherirse al concepto de kiss.
- Preguntame cualquier duda antes de lanzarse a implementar.
- No debes de modificar nada que no sea necesario.

## Manejo de git

- Debes de crear un branch con el nombre de la tarea para esta tarea.
- Debes de hacer un commit al terminar la tarea.
- Debes de hacer un push al terminar la tarea.
- Debes de hacer un pull request al terminar la tarea.

## Nomenclatura

- Los nombres deben ser descriptivos y en inglés.
- Los nombre no deben ser muy sofisticados.
- Los nombres y commits messages deben ser descriptivos y en inglés, ademas de usar la convencion de commits de git. No delates el ser una IA.
